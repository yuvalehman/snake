To run this game simply run the main python file (make sure you run the file in any py3 env).
As suggested in the instruction the user should provide the metadata for game to actually run.
Reminder from the game instructions, input order:
1. Cases
2. 'n' - size of the board
3. 'k' - number of moves / input lines during the game is being played
4. Food locations, list of tuples
5. first input line
.
.
.
5+k. last input line
